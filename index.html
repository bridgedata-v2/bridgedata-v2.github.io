BridgeData V2: A Dataset for Robot Learning at Scale

Evaluations of Offline Learning Methods

We evaluated several state-of-the-art offline learning methods using the dataset. We selected both goal-conditioned methods (in the form of goal images) and language-conditioned methods. We evaluated on tasks that test a range of abilities. Below we show videos for some of the tasks evaluated in the paper.
